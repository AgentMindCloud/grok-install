## How to Use (30 seconds)
1. Add `grok-install.yaml` at the root of your repo (copy from the [full spec](standard/spec.md) – v2.12).
2. Commit & push.
3. Post your GitHub link on X → Grok automatically replies with the blue “Install with Grok” pill + pre-install security scan + Passive Growth features.

**Pro tip:** Include the `promotion:` block to fully activate the Passive Growth Engine (auto-welcome, shareable cards, weekly highlights, trending).
