## How to Use (30 seconds)
1. Add `grok-install.yaml` at the root of your repo (use the [full spec](standard/spec.md) – v2.12).
2. Commit & push.
3. Post your GitHub link on X → Grok automatically replies with the Install pill + pre-install security scan + X-native options + Passive Growth features.

**Pro tip:** Add the new `promotion:` block + `x_native_runtime:`, `intelligence_layer:`, and `reputation:` blocks + the `credits:` section for the full Phase 11 + Phase 10 experience.
