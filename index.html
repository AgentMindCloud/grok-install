What is grok-install?

grok-install is an open standard that lets any developer make their AI agent instantly installable by anyone on X.
Just add a single grok-install.yaml file — Grok handles guided setup, deployment, and everything else.

How to add it in <5 minutes

01

Add the file

Create grok-install.yaml at the root of your repo

02

Copy an example

Use one of the ready-made v1.1 templates from the /standard folder

03

Commit & done

Grok automatically detects it and guides the user through setup
