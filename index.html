Grok Commands Reference

All official grok-install commands. Copy and use on X.

Command Example What it does Troubleshooting
@grok my agents @grok my agents Shows your private dashboard with installed agents, orchestration view, and share cards If you see placeholder agents, the command is still warming up — try @grok show my agents
@grok show me [category] agents @grok show me telegram agents Browse marketplace agents by category (telegram, discord, twitter, etc.) Add category to your YAML for better visibility
@grok orchestrate my agents @grok orchestrate my agents View and create connections between your agents (Phase 7) Requires agents with orchestration.enabled: true
@grok install this Reply to any GitHub link containing grok-install.yaml Starts one-click install flow Works on public repos with valid YAML
@grok generate grok-install.yaml @grok generate grok-install.yaml Grok reads your repo and creates the YAML file Reply to your own repo link
Voice install “Hey Grok, install Hermes Telegram Dashboard” Voice-activated installation Works when Grok is listening
