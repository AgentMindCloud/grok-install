What is grok-install?

grok-install is an open standard (v2.0) that lets any developer make their AI agent instantly installable by anyone on X.
Just add a single grok-install.yaml file — Grok handles the rest.

How to add it in <5 minutes

01

Add the file

Create grok-install.yaml at the root of your repo

02

Copy an example

Use one of the ready-made templates from the /standard/examples folder

03

Commit & done

Grok automatically detects it and offers one-click install

Simple mode vs Advanced mode

Default is Simple mode (maximum hand-holding). Switch to Advanced mode for full control.
