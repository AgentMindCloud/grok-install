Grok Commands Reference

All official grok-install commands. Copy and use on X.

Command Example What it does Troubleshooting
@grok my agents @grok my agents Shows your private dashboard with installed agents, orchestration view, and share cards If you see placeholder agents, try @grok show my agents
@grok show me [category] agents @grok show me telegram agents Browse marketplace agents by category Add category and tags to your YAML
@grok show me agents that use [API] @grok show me agents that use Tesla API Advanced filtered search Add matching tags to your YAML
@grok orchestrate my agents @grok orchestrate my agents View and create connections between your agents Requires orchestration.enabled: true
@grok update all my agents @grok update all my agents One command updates every installed agent (new in v2.9) Works on all agents with valid grok-install.yaml
@grok generate template [pattern] @grok generate template orchestration Grok creates ready-to-use orchestration templates (new in v2.9) Reply to your repo link
@grok install this Reply to any GitHub link containing grok-install.yaml Starts one-click install flow with pre-install security scan Works on public repos with valid YAML
