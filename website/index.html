grok-install Standard Templates Docs GitHub

The open standard for Grok agents on X.

A YAML manifest spec and validator that turns any AI agent into a one-click install on X.

One declarative grok-install.yaml file describes an agent's runtime, prompts, tools, safety profile, and X integration. A small Python CLI validates manifests against the v2.14 schema and a GitHub Action gates them in CI. Eighteen ready-to-fork templates cover everything from a daily brief bot to a multi-agent research swarm.
