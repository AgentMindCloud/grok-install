grok-install

One YAML. Mint an agent on X.

A YAML manifest spec, a Python validator, and a GitHub Action that turn any AI agent into a one-click install on X. Eighteen ready-to-fork templates, one locked v2.14 schema, zero glue code.

Read the spec →

Quickstart

# 1. Install the validator (from source for v1.0; PyPI in v1.1)
pip install git+https://github.com/AgentMindCloud/grok-install.git

# 2. Validate a manifest
grok-install validate path/to/grok-install.yaml

# 3. Install with the cross-platform runtime
#    See xlOS: https://github.com/AgentMindCloud/xlOS
