Manifest schema

The canonical schema for a grok-install.yaml manifest is a JSON Schema 2020-12 document. Conformant parsers accept unknown top-level keys and unknown extensions: sub-keys per the backward-compatibility policy in DECISIONS.md D5.

  • spec/v2.14/schema.json — the canonical schema
  • spec/v2.14/extensions/ — extension blocks reference
  • v2.14 docs — human-readable spec walkthrough
