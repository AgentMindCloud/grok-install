# Ecosystem

`grok-install` is a spec, not a framework. It plays well with the
broader AI-agent ecosystem.

<div class="grok-features" markdown>

<div class="grok-feature" markdown>
### [xAI SDK →](xai-sdk.md)
The default runtime. `grok-install` never re-implements the SDK — it
wraps it.
</div>

<div class="grok-feature" markdown>
### [LiteLLM →](litellm.md)
Use any provider (OpenAI, Anthropic, Ollama) behind a unified interface.
</div>

<div class="grok-feature" markdown>
### [Semantic Kernel →](semantic-kernel.md)
Bridge `grok-install` agents into the Microsoft Semantic Kernel
planner.
</div>

</div>
