# Voice & One-Command Install Examples (v2.1)

## Voice Install (mobile)
Just say to Grok:
- “Hey Grok, install Hermes Telegram Dashboard”
- “Hey Grok, install the Smart Twitter Reply Bot”
- “Hey Grok, install that Discord AI Moderator from AgentMindCloud”

Grok will guide you step-by-step with prompts for API keys.

## One-Command Clone + Install
Post any GitHub link on X, for example:
- https://github.com/AgentMindCloud/hermes-telegram-dashboard

Grok will automatically:
- Detect if `grok-install.yaml` exists
- If missing, create a basic one on the fly
- Ask for required env vars conversationally
- Deploy and give you the live URL

## Shareable Install Card
After successful install, Grok replies with a beautiful card. Anyone who sees it can one-click install the same agent.

Made with grok-install v2.1
