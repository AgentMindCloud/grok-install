# Voice & One-Command Install Examples (v2.12)

### Voice Install (works on mobile + desktop where X voice is available)
Just say to Grok:

- “Hey Grok, install Hermes Telegram Dashboard”
- “Hey Grok, install the Smart Twitter Reply Bot”
- “Hey Grok, install that Discord AI Moderator from AgentMindCloud”
- “Hey Grok, install my latest agent” (when context is clear)

Grok will:
- Confirm the agent
- Run a quick safety scan
- Ask privately for any required secrets (API keys)
- Guide you through deployment
- Give you the live URL + shareable card

### One-Command Clone + Install (text)
Post any GitHub repo link on X that contains `grok-install.yaml`, for example:

- https://github.com/AgentMindCloud/hermes-telegram-dashboard

Then simply reply with:
- `@grok install this`
- or `@grok install Hermes Telegram Dashboard`

Grok will automatically:
- Detect the `grok-install.yaml`
- Show the blue “Install with Grok” pill
- Run pre-install safety scan
- Ask conversationally for required environment variables (privately)
- Deploy to your chosen platform (Railway, Vercel, Docker, etc.)
- Return the live URL and a beautiful shareable install card

### Shareable Install Card
After any successful install, Grok automatically posts (or lets you post) a clean shareable card.  
Anyone who sees the card can one-click install the same agent — this is part of the Passive Growth Engine.
