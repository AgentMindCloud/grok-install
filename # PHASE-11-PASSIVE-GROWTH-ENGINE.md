## Core Features (ranked by priority – one big clean update per file)

### 1. Auto-Welcome for New Repos (Highest)
- Grok detects any public GitHub link containing grok-install.yaml
- Automatically replies with welcome message + blue “Install with Grok” pill + Verified by Grok badge
- Includes direct link to the agent’s shareable card

### 2. Auto-Share Successful Installs
- After every successful install, Grok posts a public shareable card in the thread
- Card shows: agent name, builder, install count, “Install with Grok” button

### 3. Auto-Weekly Highlight Thread
- Every Sunday Grok auto-posts a new thread using weekly-thread-template.md
- Shows: Top installed agents, new agents this week, trending triggers

### 4. Trending & Featured Boost
- Most-installed agents get automatic visibility in Grok replies and marketplace
- “Trending this week” section on index.html updates live

### 5. Referral & Builder Attribution
- Every shareable card credits the original builder and links back to their repo
- Automatic “Powered by grok-install” mention with @JanSol0s tag
