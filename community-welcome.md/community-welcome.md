# grok-install Builders Community

Welcome to the official grok-install builders community!

This is a low-maintenance space for developers who have added (or want to add) grok-install.yaml to their AI agents.

## Why join?
- Share your installed agents
- Get quick help with YAML setup or deployment
- See what others are building
- Weekly progress updates from the core project
- Occasional calls for beta testers

## Simple Rules (keep it clean)
1. Stay on-topic: grok-install, agent installs, Grok on X, Musk ecosystem agents
2. Be respectful and helpful
3. No spam or self-promotion without adding real value
4. Tag @JanSol0s only when you need core project feedback

## How to get started
1. Post a message like: “Just added grok-install to my [agent name] — here’s the repo: [link]”
2. Add the “Featured in Grok” badge to your README (copy from docs site)
3. Try installing one of the demo agents and share your experience
