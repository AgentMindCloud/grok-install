# Welcome to the grok-install Builders Community

This is the official low-maintenance space for developers who have added (or want to add) `grok-install.yaml` to their AI agents.

### Why Join?
- Share your installed agents and get visibility through the Passive Growth Engine
- Get quick help with YAML setup, deployment, orchestration, or voice commands
- See what others are building with Native X Runtime, Intelligence Layer, and Marketplace features
- Receive weekly progress updates and highlight threads
- Participate in occasional calls for beta testers and feedback

### Current Capabilities (v2.12 – April 2026)
- Passive Growth Engine (auto-welcome, auto-share cards, weekly highlights, trending)
- Professional Marketplace with glassmorphism design
- Full Voice & Mobile Experience (voice commands + mobile-first dashboard)
- Native X Agent Runtime, Grok Intelligence Layer, multi-agent orchestration
- Enhanced Safety & Verification 2.0 with pre-install scans

### Simple Rules (keep it clean)
1. Stay on-topic: grok-install, agent installs, Grok on X, Musk ecosystem agents
2. Be respectful and helpful
3. No spam or excessive self-promotion — add real value when sharing
4. Tag @JanSol0s only when you need core project feedback

### How to Get Started
1. Post a message like: “Just added grok-install.yaml to my [agent name] — here’s the repo: [link]”
2. Add the “Verified by Grok” or “Grok-Installed” badge to your README (copy from the docs site)
3. Try installing one of the demo agents and share your experience
4. Use voice commands on X: “Hey Grok, show me my agents” or “Hey Grok, install this”
