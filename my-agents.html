How to Use My Agents Dashboard

1. Access the dashboard
Reply to Grok on X with: @grok my agents or @grok show my agents
2. See your installed agents + analytics
All agents you have installed via grok-install will appear here with live stats.
3. Orchestrate agents
Click “Link My Agents (Orchestrate)” to connect agents (e.g. Twitter bot → Hermes dashboard).
4. Share install cards
Use the Share button to create beautiful X posts.

Tip: Make sure your agents have orchestration.enabled: true in their grok-install.yaml for full features.
