How to Use My Agents Dashboard

1. Access the dashboard
Simply reply to Grok on X with: @grok my agents
2. See your installed agents
All agents you have installed via grok-install appear here automatically.
3. Share install cards
Click “Share Install Card” to generate a beautiful postable card for X.
4. Orchestrate agents (new in Phase 7)
Click the big “Link My Agents (Orchestrate)” button below to connect agents.
Example: Link your Twitter Reply Bot to Hermes Dashboard so mentions automatically update your dashboard.
5. Community tools
Use Voting and Moderation Queue to participate in the grok-install community.

Tip: Make sure your agents have orchestration.enabled: true in their grok-install.yaml for full features.
